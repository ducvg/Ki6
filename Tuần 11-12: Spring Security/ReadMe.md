# Demo Project
Demo áp dụng Spring Security vào một ứng dụng SpringBoot

## Tính năng

- Phân quyền truy cập từng trang web
- Mã hóa mật khẩu BCrypt

## Documentation

- IDE: IntelliJ IDEA Community Edition 2023.1.1
- Database: Oracle
- Structure: MVC
- API: Spring data JPA
- Localhost với Tomcat 8.0
- JDK: Java 17
